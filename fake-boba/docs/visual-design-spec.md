# 假装喝奶茶 - 视觉设计要素清单

## 一、摇奶茶界面（制作阶段）

### 场景氛围
- **背景**：温暖的奶茶店操作台感觉，浅米色/奶油色渐变，可加淡淡的光斑
- **整体色调**：暖色系（米白、奶茶棕、淡粉）

### 核心元素

#### 1. 雪克杯（摇摇杯）
```
形状：梯形，上宽下窄（上口约 130px，底部约 90px，高度约 180px）
结构：
  - 杯盖：半圆弧形穹顶，带一个小凸起的"盖钮"，浅灰/银色金属质感
  - 杯身：透明塑料质感，能看到里面的液体和冰块
  - 杯底：略微收窄，圆弧底

材质：
  - 半透明塑料，有轻微的反光高光条（左侧竖向）
  - 边缘有细细的塑料接缝线

状态变化：
  - 静止：直立，液体平静
  - 摇晃：左右摇摆，液体翻滚，气泡上升
```

#### 2. 液体（在雪克杯内）
```
颜色：根据茶底变化
  - 红茶：#C4956A 奶茶棕
  - 绿茶：#8BC34A 抹茶绿
  - 乌龙：#D4B896 浅棕
  - 芋泥：#B39DDB 淡紫
  - 芒果：#FFD54F 金黄
  - 黑糖：#8D6E63 深棕

质感：
  - 表面有微微的波纹光泽
  - 半透明，能看到后面的小料轮廓
  - 摇晃时有气泡从底部上升
```

#### 3. 冰块（在液体中）
```
数量：2-6 块（根据用户选择）
形状：不规则多边形，边缘圆润
颜色：半透明冰蓝，带白色高光
大小：约 20-25px
动画：漂浮、轻微旋转、碰撞
```

#### 4. 小料（沉在杯底）
```
珍珠：深棕色圆球，直径约 12px，有光泽高光
芋圆：淡紫色圆球，有条纹纹理
椰果：半透明米白色方块
红豆：深红色小圆球
仙草：黑色半透明方块
布丁：金黄色椭圆形
```

#### 5. 摇晃进度环
```
位置：雪克杯下方
样式：圆环进度条
  - 外圈：浅灰色底环
  - 填充：粉橙渐变（#FF6B8A → #FFB74D）
  - 粗细：8px
  - 直径：100px

中心文字：
  - "摇一摇！"（摇晃前）
  - "35%"（摇晃中，数字递增）
  - "完成！"（100%时）
```

#### 6. 摇晃按钮
```
形状：圆角胶囊按钮
大小：约 180px × 50px
颜色：粉橙渐变背景
文字："💪 按住摇一摇！"
效果：按下时缩小，释放恢复
```

### 动画效果
- 摇晃时：雪克杯左右摆动（±15°），液体波动，气泡上升
- 完成时：杯子抖一下，显示"叮～"文字特效

## 二、倒奶茶界面（出品阶段）

### 场景布局
```
左侧：倾斜的雪克杯（正在倒出）
中间：液体流（从雪克杯流向饮用杯）
右侧：饮用杯（逐渐填满）
```

### 核心元素

#### 1. 雪克杯（倾斜状态）
```
角度：向右倾斜约 45°
状态：盖子打开（或半开），液体从杯口流出
液体：逐渐减少，液面下降
```

#### 2. 液体流
```
形状：弧形曲线，从雪克杯口流向饮用杯
颜色：与茶底颜色一致
质感：半透明，有流动的光泽
动画：持续流动，像真实倒水
```

#### 3. 饮用杯（最终成品杯）
```
形状：经典奶茶杯
  - 杯口：略宽，有杯沿
  - 杯身：略微收腰，透明塑料质感
  - 杯底：平底
  - 高度：约 160px

吸管：
  - 颜色：粉橙渐变（#FF6B8A → #FFB74D）
  - 位置：斜插在杯中，露出杯口约 40px
  - 形状：圆柱形，顶端斜切

杯内：
  - 液体：从 0% 逐渐上升到 80%
  - 冰块：漂浮在液体中
  - 小料：沉在底部
  - 奶盖（如有）：铺在液面上，白色奶油质感
```

#### 4. 背景装饰
```
可选元素：
  - 蒸汽（热饮）：从杯口升起的淡淡白雾
  - 光斑：温暖的背景光晕
```

### 动画效果
- 雪克杯倾斜 → 液体流出 → 饮用杯液面上升
- 完成后：雪克杯淡出，饮用杯居中放大

## 三、喝奶茶界面（品尝阶段）

### 场景氛围
- **背景**：温馨放松，可加淡淡的暖色光晕
- **整体感觉**：满足、享受

### 核心元素

#### 1. 饮用杯（完整展示）
```
同"倒奶茶界面"的饮用杯，但：
  - 液面高度：从 80% 开始，每喝一口下降 16%
  - 吸管：用户点击时，吸管底部有漩涡
  - 杯身：喝到第3口后，杯身微微倾斜（仰头喝的感觉）
```

#### 2. 吸管细节
```
材质：塑料吸管，有光泽
动态：
  - 喝的时候：吸管底部产生漩涡
  - 吸管口：有小气泡或液体流动的暗示
```

#### 3. 漩涡特效（喝的时候）
```
位置：吸管底部，液体表面
形状：螺旋形漩涡
颜色：液体颜色的深色版
动画：旋转 2-3 圈后消失
```

#### 4. 飘出的星星/爱心
```
触发：每喝一口
形状：交替出现星星 ⭐ 和爱心 ❤️
颜色：粉红色 #FF6B8A
动画：从吸管口飘出，向上飞，逐渐消失
```

#### 5. 情绪气泡
```
位置：杯子旁边
形状：圆角对话气泡
内容：随机显示
  - "嗯~ 好满足！"
  - "珍珠QQ的！"
  - "快乐在冒泡~"
  - "再来一口！"
动画：弹出 → 停留 1.5 秒 → 淡出
```

#### 6. 进度指示
```
位置：杯子下方
样式：5 个小圆点
状态：
  - 未喝：空心圆圈
  - 已喝：实心填充（粉橙渐变）
文字："第 X / 5 口"
```

#### 7. 喝完特效
```
杯子：倾斜 15°（喝光的样子）
笑脸：在杯子上方浮现一个可爱的笑脸 😊
纸屑：彩色纸屑从上方飘落
文字："好喝！下次再来～"
```

#### 8. 暖色遮罩（第3口后）
```
效果：整个画面叠加一层淡淡的暖橙色
透明度：约 10-15%
目的：营造满足、温暖的感觉
```
